Política de Privacidad del Dashboard de Publicidad

Esta política de privacidad describe cómo se maneja la información en el Dashboard de Publicidad.

1. Recopilación de Información

Este dashboard no recopila, almacena ni comparte información personal de los usuarios. Todas las credenciales de API se almacenan localmente en el servidor del usuario y no son accesibles por el desarrollador.

Los datos de las campañas publicitarias se obtienen directamente de las APIs de las plataformas (Google Ads, Facebook Ads, TikTok Ads) y solo se utilizan para su visualización en el dashboard. Estos datos no son almacenados permanentemente ni compartidos con terceros.

2. Uso de la Información

La información obtenida a través de las APIs se utiliza exclusivamente para generar los reportes y gráficos que componen el dashboard, permitiendo al usuario monitorear el rendimiento de sus campañas.

3. Compartición de Información

Ninguna información (ni personal ni de campañas) es compartida con terceros por parte de este dashboard.

4. Seguridad

La seguridad de las credenciales de API y los datos obtenidos depende de la configuración del servidor donde se ejecuta el dashboard y de las prácticas de seguridad del usuario.

Fecha de última actualización: 3 de junio de 2025
